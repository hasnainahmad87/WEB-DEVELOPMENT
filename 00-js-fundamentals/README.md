# JavaScript Fundamentals

A single-file JavaScript reference and practice document covering the four core pillars of the language: data types, functions, control flow, and arrays & loops.

## 📋 Topics Covered

| Section | Concepts |
|---------|----------|
| Data Types | Primitives, typeof, type coercion, template literals |
| Functions & Methods | Declarations, arrow functions, default/rest params, string/number methods |
| Control Flow | if/else, ternary, switch, logical operators |
| Arrays & Loops | push/pop/filter/map/reduce, for/for-of/while, spread, destructuring |

## 📂 Files

```
00-js-fundamentals/
└── fundamentals.js     # All exercises with comments and console.log output
```

## ▶️ Running

Open in any browser console, or run with Node.js:

```bash
node fundamentals.js
```

## 🎯 Learning Outcomes

- Understand JavaScript's type system and how coercion works
- Write reusable functions using modern ES6+ syntax
- Use conditional logic to make decisions in programs
- Transform and iterate over data using array methods

---

*Foundational exercises written as part of learning vanilla JavaScript before moving on to frameworks.*
