## 🚀 Running the Projects

Most HTML/CSS/JS projects open directly in a browser. Projects with build steps include their own README with setup instructions.

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/web-development-portfolio.git

# For React (Memory Game)
cd 03-memory-card-game && npm install && npm run dev

# For Vue (Quiz App)
cd 04-quiz-app && npm install && npm run dev

# For Banking Dashboard (needs Node.js API)
cd 06-banking-dashboard/api && npm install && node server.js
```
