# Pixel Painter — Keyboard Game

Author: Hasnain Ahmad <hasnainswati2004@gmail.com>

Controls
- Arrow keys: move cursor
- Shift + Arrow: jump 3 cells
- Space: paint current cell
- R/G/B/Y/K: change color
- P: toggle paint-mode
- Enter: start/reset level

Features
- Uses three keyboard events: `keydown`, `keypress`, and `keyup`.
- Visual feedback: highlighted cursor, colored cells, faint target overlay.
- Game logic: score for matching target pixels, levels advance when you complete target.

Open `index.html` in a browser to play.
